Query

Source:
  • url_variables.js, line 6
A query string, an url or an object
Type:
  • String | Object
Examples
"a=1&b=2&c=3"	// query string
{"a":1, "b":2, "c":3}	// object
{"a":[1, 2], "b":2, "c":3}	// object
"http://example.com"	// an url without queries
"http://example.com?a=1&b=2&c=3"	// an url with queries
